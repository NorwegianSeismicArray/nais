# Pretrained Models

This repository will contain pretrained models specifically to NORSAR. 

## Spectrograms
Current available pretrained spectrogram models and data used for training: 
* AutoEncoder
  * GeobyIT Alna data, Raspberry shakes. Noisy with few events.
